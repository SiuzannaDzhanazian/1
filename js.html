Я стану фронтенд-разработчиком

Фронтенд-разработчик создает удобные интерфейсы на сайте для того,чтобы пользователи могли комфорт им пользоваться

Что нужно знать фронту?

  1. HTML
  2. CSS
  3. JS

HTML - это язык разметки страницы, он используется для создания структуры сайта, проще говоря, скелета

CSS - это язык стилей, нужен для стилизации наших тегов т.е. сделать страницу красивой

JS - это язык программирования, он дает интерактивности нашей странице
